**What was built:** Complete vertical swipe video feed with preloading, buffering indicator, and end-of-feed state. This is the full Phase 3 deliverable.

**Desktop browser (Chrome/Firefox):**
1. Open http://localhost:5173
2. First video should autoplay muted
3. Scroll mousewheel down — should advance to next video (FEED-01, D-03)
4. Scroll mousewheel up — should return to previous video (FEED-02)

8. Scroll to the last video, then one more — should see "You've seen everything" on black (FEED-06, D-08)
9. Try scrolling further — should be blocked (D-09)
10. Scroll back up — should return to last video (D-09 back-swipe allowed)

**Mobile device (iOS Safari preferred):**
11. Open the LAN URL shown by Vite (e.g., http://192.168.x.x:5173)
12. First video autoplays muted, fills screen without letterboxing
13. Swipe up — next video snaps cleanly, no overshoot on fast flick (FEED-03)
14. Swipe down — previous video returns
15. Verify video transitions feel TikTok-like — decisive snap, not floaty (D-01)
16. Tap mute button, then swipe — sound persists across videos (D-05)

**Buffering test (optional — requires network throttle):**
17. Open DevTools > Network > Throttle to "Slow 3G"
18. Reload — should see white spinner centered on black while video buffers (FEED-05, D-06)
